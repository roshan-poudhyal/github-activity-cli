# GitHub Activity CLI

A simple Node.js command-line tool that fetches and displays a GitHub user's recent activity using the GitHub API. It also supports pagination to view older events.

---

## Features

- Fetch GitHub user activity from API
- Display recent events in terminal
- Supports pagination (view older activity pages)
- Simple CLI interface
- Built using only Node.js core modules (no external libraries)

---

## Project Structure

github-activity-cli/
├── index.js
├── githubService.js

## Requirements

- Node.js installed (v14+ recommended)
